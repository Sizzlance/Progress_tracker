# Pro-Tracker: Gym Progress Dashboard

Веб-приложение для отслеживания тренировочного прогресса и углубленной аналитики объема нагрузок. Построено на Python с использованием Streamlit и Pandas.

## Функционал
* **Гибкая настройка программы**: Создание собственных тренировочных дней (например, гибридный сплит Push/Pull/Legs или Upper/Lower) и привязка к ним пула упражнений.
* **Умное логирование**: Динамические формы для записи рабочих подходов. Удобно фиксировать тяжелые сеты (например, на 6-8 повторений) и отслеживать прогрессию весов.
* **Аналитика и графики**: Использование Pandas для агрегации данных. Автоматический расчет общего тоннажа, среднего рабочего веса (интенсивности) и визуализация прогресса по конкретным упражнениям.

## Технологии
* **Frontend**: Streamlit
* **Backend**: Python, SQLite3 (хранение логов, структуры тренировок)
* **Data Science**: Pandas (обработка датафреймов, группировка статистики)
